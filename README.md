# TR26-speedometer
Repo for Arduino and RaceCapture code for making the speedometer of the Terps Racing Baja Bug car work with CAN communication. 
Used with RaceCapture Pro Mk3, Arduino Uno R3, and a TJA1050 CAN Module Board with an MCP2515 CAN controller.

ASSOCIATED COMPONENTS:
- RaceCapture Pro Mk3 (which should have a GPS sensor with it): https://wiki.autosportlabs.com/RaceCapture-Pro_MK3
- Arduino Uno R3: https://store-usa.arduino.cc/products/arduino-uno-rev3?srsltid=AfmBOoopsUTrPjWo5VV1hiclzeiimU8moED43kgsSk4khB2XKL4m2e1O
- TJA1050 CAN Module Board (with MCP2515 CAN controller): https://www.amazon.com/dp/B07J9KZ4L4?psc=1&ref=ppx_yo2ov_dt_b_product_details
- 1602 LCD Screen w/ SPI interface: https://www.amazon.com/dp/B0B76Z83Y4?ref_=ppx_hzsearch_conn_dt_b_fed_asin_title_1&th=1

ASSOCIATED SOFTWARE:
- Arduino IDE: https://www.arduino.cc/en/software
- RaceCapture App: https://podium.live/software
- Arduino MCP2515 CAN Interface Library: https://github.com/autowp/arduino-mcp2515/tree/master
